About

I work as a Paid Search Executive managing Google Ads campaigns across a range of client accounts. Over the past year I've been moving increasingly into the technical side of the role - building integrations that connect Google Ads, GA4, and Slack together through AI-powered workflows using the Model Context Protocol (MCP).

The automation work started out of frustration with the amount of repetitive, manual work involved in account management - checking budgets, dismissing recommendations, pulling reports. I started writing small tools to handle these tasks and got hooked on the broader problem of making programmatic access to Google Ads data more useful and flexible.

Outside of work I'm building my own Google Ads dashboard and MCP server using the Google Ads API. The goal is a lightweight tool for querying campaign data, monitoring account health, and surfacing performance anomalies - something more purpose-built than the standard UI for the way I actually work with accounts.

Projects

OneClickDismiss Live

A Chrome extension that dismisses all Google Ads recommendations on the page in one click. Built because the default flow of dismissing recommendations one by one, with a confirmation dialog each time, adds up to a meaningful time drain across large accounts. The extension injects a button into the recommendations page that handles the entire sequence automatically.

Google Ads Dashboard & MCP Server In development

A personal project - building a custom Google Ads management dashboard and MCP server backed directly by the Google Ads API. The aim is a lightweight interface for campaign monitoring, performance reporting, and keyword planning that works the way I actually use data, rather than the way the standard UI presents it.

Planned API usage: GoogleAdsService for campaign performance queries and reporting, CampaignService for account structure and status monitoring, KeywordPlanService for keyword research tooling, and CustomerService for account hierarchy management. Initial scope is my own Google Ads accounts.
